# 🧠 Cryptocurrency Portfolio Manager
The Cryptocurrency Portfolio Manager is a application designed to help with their cryptocurrency investments. It provides a robust platform for tracking portfolio performance. The application is built using a variety of technologies, including Java, and integrates with the CoinMarketCap API to fetch real-time market data.

## 🚀 Features
- **Portfolio Management**: Create, read, update, and delete portfolio entries to track your cryptocurrency investments.
- **Market Data Integration**: Fetch real-time market data from the CoinMarketCap API to stay up-to-date with the latest cryptocurrency prices and trends.
- **Data Encryption**: Utilize cryptographic functionality to secure your portfolio data and ensure confidentiality.
- **Configuration Management**: Load and manage configuration settings to customize the application to your needs.

## 🛠️ Tech Stack
* Java
* CoinMarketCap API
* Database connection libraries (JDBC)
* HTTP client libraries
* JSON parsing libraries

## 📦 Installation
### Prerequisites
* Java Development Kit (JDK) 8 or later
* Maven or Gradle for building and dependency management
* A compatible database management system (e.g., MySQL, PostgreSQL)
